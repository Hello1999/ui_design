## 色彩基础

### 色彩的三大属性

#### 1. 色相 (Hue)

**定义：** 颜色的基本类型，即我们通常说的"什么颜色"

```
基本色相：
- 红色 (Red)
- 橙色 (Orange)
- 黄色 (Yellow)
- 绿色 (Green)
- 蓝色 (Blue)
- 紫色 (Purple)

色相环：
红 → 橙 → 黄 → 绿 → 青 → 蓝 → 紫 → 红
```

#### 2. 饱和度 (Saturation)

**定义：** 颜色的纯度或鲜艳程度

```
高饱和度：
- 鲜艳、生动
- 吸引注意力
- 适合强调元素
示例：#FF0000（纯红色）

低饱和度：
- 柔和、中性
- 不刺激
- 适合背景
示例：#CCCCCC（灰色）

无饱和度：
- 灰色系
```

```css
/* 饱和度示例 */
.high-saturation {
  background: hsl(200, 100%, 50%); /* 完全饱和的蓝色 */
}

.medium-saturation {
  background: hsl(200, 50%, 50%); /* 中等饱和度 */
}

.low-saturation {
  background: hsl(200, 20%, 50%); /* 低饱和度，偏灰 */
}

.no-saturation {
  background: hsl(200, 0%, 50%); /* 无饱和度，纯灰色 */
}
```

#### 3. 明度 (Lightness/Brightness)

**定义：** 颜色的明暗程度

```
高明度：
- 接近白色
- 轻盈、清新
- 适合背景

中明度：
- 平衡
- 易于阅读
- 通用性强

低明度：
- 接近黑色
- 严肃、稳重
- 适合文本
```

```css
/* 明度示例 */
.light {
  background: hsl(200, 70%, 90%); /* 浅蓝色 */
}

.medium {
  background: hsl(200, 70%, 50%); /* 中等蓝色 */
}

.dark {
  background: hsl(200, 70%, 20%); /* 深蓝色 */
}
```

## 色彩模型

### 1. RGB 模型

**用途：** 屏幕显示（加色模型）

```
组成：
- Red（红）：0-255
- Green（绿）：0-255
- Blue（蓝）：0-255

示例：
rgb(59, 130, 246)  // 蓝色
rgb(239, 68, 68)   // 红色
rgb(16, 185, 129)  // 绿色
```

```css
.rgb-color {
  /* RGB 标准格式 */
  color: rgb(59, 130, 246);

  /* RGB with Alpha（透明度）*/
  background: rgba(59, 130, 246, 0.5); /* 50% 透明度 */
}
```

### 2. HEX 模型

**用途：** Web 开发最常用

```
格式：#RRGGBB
- RR：红色值（00-FF）
- GG：绿色值（00-FF）
- BB：蓝色值（00-FF）

示例：
#3B82F6  // 蓝色（完整格式）
#F00     // 红色（简写格式 = #FF0000）
#10B981  // 绿色

带透明度：
#3B82F680  // 蓝色 50% 透明（80 = 50% in hex）
```

```css
.hex-color {
  color: #3B82F6;
  background: #F3F4F6;

  /* 8 位 HEX（包含透明度）*/
  border-color: #3B82F680; /* 最后两位 80 = 50% alpha */
}
```

### 3. HSL 模型

**用途：** 设计师友好，易于调整

```
组成：
- Hue（色相）：0-360度
- Saturation（饱和度）：0-100%
- Lightness（明度）：0-100%

优势：
✓ 直观理解颜色
✓ 易于创建变体
✓ 便于调整明暗
```

```css
:root {
  /* 基础颜色 */
  --primary-hue: 217;
  --primary-sat: 91%;

  /* 创建色阶 */
  --primary-50: hsl(var(--primary-hue), var(--primary-sat), 95%);
  --primary-100: hsl(var(--primary-hue), var(--primary-sat), 90%);
  --primary-500: hsl(var(--primary-hue), var(--primary-sat), 60%);
  --primary-900: hsl(var(--primary-hue), var(--primary-sat), 20%);
}

/* 使用 */
.button {
  background: var(--primary-500);
}

.button:hover {
  /* 只改变明度，创建悬停效果 */
  background: hsl(var(--primary-hue), var(--primary-sat), 55%);
}
```

**HSL 实用技巧：**

```css
/* 快速创建颜色变体 */
.original {
  background: hsl(200, 70%, 50%);
}

/* 变浅：提高明度 */
.lighter {
  background: hsl(200, 70%, 70%);
}

/* 变深：降低明度 */
.darker {
  background: hsl(200, 70%, 30%);
}

/* 降低饱和度（更中性）*/
.muted {
  background: hsl(200, 30%, 50%);
}

/* 改变色相（不同颜色）*/
.different {
  background: hsl(280, 70%, 50%); /* 紫色 */
}
```

### 模型转换工具

```javascript
// RGB 转 HEX
function rgbToHex(r, g, b) {
  return "#" + [r, g, b]
    .map(x => x.toString(16).padStart(2, '0'))
    .join('');
}

// HEX 转 RGB
function hexToRgb(hex) {
  const result = /^#?([a-f\d]{2})([a-f\d]{2})([a-f\d]{2})$/i.exec(hex);
  return result ? {
    r: parseInt(result[1], 16),
    g: parseInt(result[2], 16),
    b: parseInt(result[3], 16)
  } : null;
}
```

## 配色方案

### 1. 单色配色 (Monochromatic)

**定义：** 使用同一色相的不同明度和饱和度

```
特点：
✓ 和谐统一
✓ 简洁优雅
✓ 易于掌握
✓ 专业感强

适用：
- 简约设计
- 企业网站
- 仪表盘
```

```css
/* 蓝色单色配色 */
:root {
  --primary-50: #EFF6FF;   /* 极浅 */
  --primary-100: #DBEAFE;
  --primary-200: #BFDBFE;
  --primary-300: #93C5FD;
  --primary-400: #60A5FA;
  --primary-500: #3B82F6;  /* 主色 */
  --primary-600: #2563EB;
  --primary-700: #1D4ED8;
  --primary-800: #1E40AF;
  --primary-900: #1E3A8A;  /* 极深 */
}

/* 使用 */
.background {
  background: var(--primary-50);
}

.card {
  background: var(--primary-100);
  border: 1px solid var(--primary-200);
}

.button {
  background: var(--primary-500);
  color: white;
}

.button:hover {
  background: var(--primary-600);
}

.text-muted {
  color: var(--primary-700);
}
```

### 2. 类似色配色 (Analogous)

**定义：** 色轮上相邻的颜色（通常 30-60 度范围）

```
特点：
✓ 和谐自然
✓ 丰富多彩
✓ 温暖或冷色调一致

示例：
- 蓝色 + 青色 + 绿色
- 红色 + 橙色 + 黄色
- 紫色 + 蓝色 + 青色
```

```css
/* 蓝-青-绿类似色 */
:root {
  --color-blue: #3B82F6;    /* 蓝 */
  --color-cyan: #06B6D4;    /* 青 */
  --color-teal: #14B8A6;    /* 青绿 */
}

/* 应用：渐变背景 */
.gradient-analogous {
  background: linear-gradient(
    135deg,
    var(--color-blue),
    var(--color-cyan),
    var(--color-teal)
  );
}
```

### 3. 互补色配色 (Complementary)

**定义：** 色轮上相对的颜色（180 度）

```
特点：
✓ 高对比度
✓ 充满活力
✓ 吸引注意力
⚠️ 使用需谨慎，避免过于刺激

互补色对：
- 蓝色 ↔ 橙色
- 红色 ↔ 绿色
- 黄色 ↔ 紫色
```

```css
/* 蓝-橙互补色 */
:root {
  --primary: #3B82F6;      /* 蓝色 */
  --accent: #F97316;       /* 橙色 */
}

/* 使用建议：一主一辅 */
.page {
  background: #FFFFFF;
}

.heading {
  color: var(--primary);   /* 主色用于主要元素 */
}

.cta-button {
  background: var(--accent); /* 互补色用于强调 */
  color: white;
}

/* ⚠️ 避免大面积同时使用 */
.bad-example {
  background: var(--primary);
  color: var(--accent);     /* 太刺眼！ */
}
```

### 4. 分离互补色 (Split-Complementary)

**定义：** 一个基础色 + 其互补色两侧的颜色

```
特点：
✓ 比互补色更柔和
✓ 保持对比度
✓ 更容易平衡

示例：
基础色：蓝色
分离互补：橙黄色 + 橙红色
```

```css
:root {
  --base: #3B82F6;         /* 蓝色 */
  --accent-1: #F59E0B;     /* 橙黄 */
  --accent-2: #EF4444;     /* 橙红 */
}
```

### 5. 三角配色 (Triadic)

**定义：** 色轮上均匀分布的三种颜色（120 度间隔）

```
特点：
✓ 鲜艳平衡
✓ 充满活力
✓ 适合多彩设计

示例：
- 红 + 黄 + 蓝（三原色）
- 橙 + 绿 + 紫（三间色）
```

```css
/* 橙-绿-紫三角配色 */
:root {
  --triad-1: #F97316;      /* 橙色 */
  --triad-2: #10B981;      /* 绿色 */
  --triad-3: #8B5CF6;      /* 紫色 */
}

/* 使用：一个主色，两个辅助色 */
.primary {
  color: var(--triad-1);
}

.secondary {
  color: var(--triad-2);
}

.tertiary {
  color: var(--triad-3);
}
```

### 6. 方形配色 (Tetradic/Square)

**定义：** 两对互补色（色轮上均匀分布的四种颜色）

```
特点：
✓ 颜色丰富
✓ 难以平衡
✓ 适合复杂设计

⚠️ 需要经验才能用好
```

## UI 配色实践

### 60-30-10 规则

**黄金配色比例**

```
60% - 主色（Primary）
用于：背景、大面积区域
特点：中性、不刺激

30% - 次色（Secondary）
用于：卡片、次要区域
特点：支持主色，增加层次

10% - 强调色（Accent）
用于：按钮、链接、重要元素
特点：高对比，吸引注意
```

```html
<div style="background: #F9FAFB; padding: 40px;"> <!-- 60% 主色 -->
  <div style="background: white; padding: 30px; border-radius: 12px;"> <!-- 30% 次色 -->
    <h2 style="color: #111827;">标题</h2>
    <p style="color: #6B7280;">这是内容区域...</p>
    <button style="background: #3B82F6; color: white; padding: 12px 24px;"> <!-- 10% 强调色 -->
      立即行动
    </button>
  </div>
</div>
```

### 深色模式配色

```css
/* 浅色模式 */
:root {
  --bg-primary: #FFFFFF;
  --bg-secondary: #F9FAFB;
  --text-primary: #111827;
  --text-secondary: #6B7280;
  --border: #E5E7EB;
}

/* 深色模式 */
@media (prefers-color-scheme: dark) {
  :root {
    --bg-primary: #111827;
    --bg-secondary: #1F2937;
    --text-primary: #F9FAFB;
    --text-secondary: #D1D5DB;
    --border: #374151;
  }
}

/* 使用 */
body {
  background: var(--bg-primary);
  color: var(--text-primary);
}

.card {
  background: var(--bg-secondary);
  border: 1px solid var(--border);
}
```

**深色模式注意事项：**

```
✓ 不要使用纯黑 (#000000)
  推荐：#111827, #1F2937
  原因：减少眼睛疲劳

✓ 降低白色文本的不透明度
  纯白：#FFFFFF
  推荐：rgba(255, 255, 255, 0.87)

✓ 调整颜色饱和度
  浅色模式：高饱和度
  深色模式：降低饱和度 10-20%

✓ 增加阴影对比
  浅色模式：box-shadow: 0 2px 8px rgba(0,0,0,0.1)
  深色模式：box-shadow: 0 2px 8px rgba(0,0,0,0.5)
```

## 色彩心理学

不同颜色会引发不同的情感和联想：

### 红色 (Red)

```
情感：
✓ 激情、能量、力量
✓ 紧迫、警告、危险
✓ 爱、兴奋

适用场景：
- 行动号召按钮
- 紧急通知
- 错误提示
- 销售、促销

避免：
- 大面积使用（过于刺激）
- 金融、医疗主色（太激进）
```

```css
.error-message {
  color: #EF4444;
  background: #FEE2E2;
}

.cta-button {
  background: #DC2626; /* 吸引点击 */
}
```

适用场景：
- 企业网站
- 金融应用
- 医疗保健
- 科技产品

## 色彩可访问性

### WCAG 对比度标准

```
AA 级（最低标准）：
- 正常文本（<18px）：至少 4.5:1
- 大文本（≥18px 或 ≥14px 粗体）：至少 3:1

AAA 级（增强标准）：
- 正常文本：至少 7:1
- 大文本：至少 4.5:1

图标和交互元素：
- 至少 3:1
```

## 总结

### 核心要点

**色彩基础**
- 色相、饱和度、明度三大属性
- RGB、HEX、HSL 三种模型
- HSL 最适合 UI 设计调整

**配色方案**
- 单色：和谐统一
- 类似色：自然丰富
- 互补色：高对比度
- 60-30-10 黄金比例

**实用原则**
- 建立语义化颜色系统
- 确保可访问性（对比度 ≥ 4.5:1）
- 考虑色盲用户
- 深色模式适配

**色彩心理**
- 蓝色：信任、专业
- 绿色：成功、自然
- 红色：紧急、警告
- 根据行业选择合适颜色
