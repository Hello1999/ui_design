## 如何使用本教材

1. **按顺序学习**：章节之间有逻辑递进关系，建议从第 1 章开始
2. **动手练习**：每章末尾有实践作业，必须亲自完成
3. **配合工具**：第 4 章之后配合 Figma 实操
4. **建立作品集**：从第 3 章起每个练习都保存为作品集素材

## 学习路径建议

```
新手入门（第 1-4 章）
    ↓
视觉基础（第 5-8 章）
    ↓
系统化设计（第 9-11 章）
    ↓
实战验证（第 12-13 章）
    ↓
求职准备（第 14 章）
```
