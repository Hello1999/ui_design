# 第五章：布局与网格系统

布局是UI设计的骨架，决定了界面元素的组织方式和视觉结构。优秀的布局系统能够创造秩序、建立层级、引导视线，并确保设计的一致性和可扩展性。本章将深入探讨现代Web布局技术和网格系统的设计与应用。

## 5.1 布局基础理论

### 5.1.1 布局的核心原则

```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <title>布局原则示例</title>
    <style>
        /* 1. 对齐（Alignment） */
        .alignment-demo {
            display: grid;
            gap: 20px;
        }

        .aligned-content {
            /* 元素对齐创造视觉秩序 */
            display: flex;
            flex-direction: column;
            align-items: flex-start; /* 左对齐 */
        }

        /* 2. 亲密性（Proximity） */
        .proximity-demo {
            /* 相关元素靠近，不相关元素分离 */
        }

        .group-related {
            margin-bottom: 2rem; /* 组之间距离大 */
        }

        .group-related > * + * {
            margin-top: 0.5rem; /* 组内元素距离小 */
        }

        /* 3. 重复（Repetition） */
        .repetition-demo {
            /* 重复的视觉元素创造一致性 */
        }

        .card-grid {
            display: grid;
            grid-template-columns: repeat(3, 1fr);
            gap: 24px;
        }

        .card {
            /* 所有卡片使用相同的样式 */
            padding: 24px;
            border-radius: 8px;
            background: white;
            box-shadow: 0 2px 8px rgba(0, 0, 0, 0.1);
        }

        /* 4. 对比（Contrast） */
        .contrast-demo {
            /* 大小、颜色、形状的对比创造层级 */
        }

        .hero-title {
            font-size: 3rem;
            font-weight: 700;
            color: #1a1a1a;
        }

        .body-text {
            font-size: 1rem;
            font-weight: 400;
            color: #666;
        }

        /* 5. 空白（White Space） */
        .whitespace-demo {
            /* 适当的空白提升可读性和视觉舒适度 */
            padding: 3rem;
            line-height: 1.8;
        }

        .breathing-room {
            margin: 2rem 0;
        }

        /* 6. 层级（Hierarchy） */
        .hierarchy-demo {
            /* 通过大小、颜色、位置建立视觉层级 */
        }

        .primary-element {
            font-size: 2rem;
            font-weight: 700;
            color: #1a1a1a;
            margin-bottom: 1rem;

        .secondary-element {

        }

        .tertiary-element {

            color: #666;
        }

        /* 7. 平衡（Balance） */
        .balance-demo {
            /* 元素的视觉重量分布均衡 */
        }

        .symmetric-layout {
            /* 对称平衡 */
            display: grid;
            grid-template-columns: 1fr 1fr;
            gap: 2rem;
        }

        .asymmetric-layout {
            /* 非对称平衡 */
            display: grid;
            grid-template-columns: 2fr 1fr;
            gap: 2rem;
        }
    </style>
</head>
<body>
    <!-- 布局原则的实际应用 -->

</html>
```

## 5.2 Flexbox 布局

### 5.2.1 Flexbox 基础

```css
/* Flexbox 容器属性 */
.flex-container {
    display: flex;

    /* 主轴方向 */
    flex-direction: row;          /* 默认：水平 */
    /* flex-direction: column;       垂直 */
    /* flex-direction: row-reverse;   水平反向 */
    /* flex-direction: column-reverse; 垂直反向 */

    /* 换行 */
    flex-wrap: nowrap;            /* 默认：不换行 */
    /* flex-wrap: wrap;               换行 */
    /* flex-wrap: wrap-reverse;       反向换行 */

    /* 简写：flex-flow = flex-direction + flex-wrap */
    flex-flow: row wrap;

    /* 主轴对齐 */
    justify-content: flex-start;  /* 默认：起始对齐 */
    /* justify-content: flex-end;     结束对齐 */
    /* justify-content: center;        居中对齐 */
    /* justify-content: space-between; 两端对齐 */
    /* justify-content: space-around;  环绕对齐 */
    /* justify-content: space-evenly;  均匀对齐 */

    /* 交叉轴对齐 */
    align-items: stretch;         /* 默认：拉伸 */
    /* align-items: flex-start;       起始对齐 */
    /* align-items: flex-end;         结束对齐 */
    /* align-items: center;            居中对齐 */
    /* align-items: baseline;          基线对齐 */

    /* 多行交叉轴对齐 */
    align-content: stretch;       /* 默认：拉伸 */
    /* align-content: flex-start;     起始对齐 */
    /* align-content: flex-end;       结束对齐 */
    /* align-content: center;          居中对齐 */
    /* align-content: space-between;   两端对齐 */
    /* align-content: space-around;    环绕对齐 */

    /* 间距（现代浏览器） */
    gap: 1rem;                    /* 所有间距 */
    row-gap: 1rem;                /* 行间距 */
    column-gap: 1rem;             /* 列间距 */
}

/* Flexbox 子项属性 */
.flex-item {
    /* 伸展因子（扩展） */
    flex-grow: 0;                 /* 默认：不扩展 */

    /* 收缩因子 */
    flex-shrink: 1;               /* 默认：允许收缩 */

    /* 基础大小 */
    flex-basis: auto;             /* 默认：根据内容 */

    /* 简写：flex = flex-grow + flex-shrink + flex-basis */
    flex: 1;                      /* 等同于 1 1 0% */
    /* flex: auto;                   等同于 1 1 auto */
    /* flex: none;                   等同于 0 0 auto */

    /* 单独的交叉轴对齐 */
    align-self: auto;             /* 默认：继承容器 */
    /* align-self: flex-start;      起始对齐 */
    /* align-self: flex-end;        结束对齐 */
    /* align-self: center;           居中对齐 */
    /* align-self: baseline;         基线对齐 */
    /* align-self: stretch;          拉伸 */

    /* 排序 */
    order: 0;                     /* 默认：按源顺序 */
}
```

## 5.3 Grid 布局

### 5.3.1 Grid 基础

```css
/* Grid 容器属性 */
.grid-container {
    display: grid;

    /* 定义列 */
    grid-template-columns: 1fr 1fr 1fr;      /* 3列等宽 */
    /* grid-template-columns: 200px 1fr 200px;  固定+弹性 */
    /* grid-template-columns: repeat(3, 1fr);   重复3次 */
    /* grid-template-columns: repeat(auto-fill, minmax(200px, 1fr)); 自适应 */

    /* 定义行 */
    grid-template-rows: auto;                /* 自动高度 */
    /* grid-template-rows: 100px 200px auto;   混合高度 */
    /* grid-template-rows: repeat(3, 100px);   重复3次 */

    /* 定义区域 */
    grid-template-areas:
        "header header header"
        "sidebar main main"
        "footer footer footer";

    /* 间距 */
    gap: 1rem;                               /* 所有间距 */
    row-gap: 1rem;                           /* 行间距 */
    column-gap: 1rem;                        /* 列间距 */

    /* 对齐整个网格 */
    justify-content: start;                  /* 主轴对齐 */
    /* justify-content: end | center | stretch | space-between | space-around | space-evenly */

    align-content: start;                    /* 交叉轴对齐 */
    /* align-content: end | center | stretch | space-between | space-around | space-evenly */

    /* 对齐网格项 */
    justify-items: stretch;                  /* 项的主轴对齐 */
    /* justify-items: start | end | center | stretch */

    align-items: stretch;                    /* 项的交叉轴对齐 */
    /* align-items: start | end | center | stretch | baseline */

    /* 简写 */
    place-content: center;                   /* align-content + justify-content */
    place-items: center;                     /* align-items + justify-items */

    /* 自动行/列大小 */
    grid-auto-rows: 100px;                   /* 自动生成行的大小 */
    grid-auto-columns: 100px;                /* 自动生成列的大小 */

    /* 自动流动方向 */
    grid-auto-flow: row;                     /* 默认：按行填充 */
    /* grid-auto-flow: column;                 按列填充 */
    /* grid-auto-flow: dense;                  密集填充（填补空隙） */
}

/* Grid 子项属性 */
.grid-item {
    /* 列位置 */
    grid-column-start: 1;                    /* 起始列 */
    grid-column-end: 3;                      /* 结束列 */
    grid-column: 1 / 3;                      /* 简写：从列1到列3 */
    /* grid-column: 1 / span 2;               简写：从列1跨2列 */

    /* 行位置 */
    grid-row-start: 1;                       /* 起始行 */
    grid-row-end: 3;                         /* 结束行 */
    grid-row: 1 / 3;                         /* 简写：从行1到行3 */
    /* grid-row: 1 / span 2;                  简写：从行1跨2行 */

    /* 区域 */
```

## 5.6 常见错误与最佳实践

### 错误1：过度嵌套

```css
/* ❌ 错误：不必要的嵌套 */
.bad-nesting {
    display: flex;
}

.bad-nesting .wrapper {
    display: flex;
    width: 100%;
}

.bad-nesting .wrapper .container {
    display: flex;
    width: 100%;
}

.bad-nesting .wrapper .container .content {
    flex: 1;
}

/* ✅ 正确：扁平化结构 */
.good-layout {
    display: flex;
}

.good-layout-content {
    flex: 1;
}
```

### 错误2：固定高度

```css
/* ❌ 错误：使用固定高度 */
.bad-height {
    height: 500px; /* 内容可能溢出或留白 */
}

/* ✅ 正确：使用最小高度或让内容决定高度 */
.good-height {
    min-height: 500px; /* 或完全不设置高度 */
}
```

### 错误3：忽略响应式

```css
/* ❌ 错误：只考虑桌面端 */
.bad-responsive {
    display: grid;
    grid-template-columns: 250px 1fr 250px;
    /* 在移动设备上会很糟糕 */
}

/* ✅ 正确：移动优先的响应式设计 */
.good-responsive {
    display: grid;
    grid-template-columns: 1fr;
    gap: 1rem;
}

@media (min-width: 768px) {
    .good-responsive {
        grid-template-columns: 200px 1fr;
    }
}

@media (min-width: 1024px) {
    .good-responsive {
        grid-template-columns: 250px 1fr 250px;
    }
}
```

### 错误4：不一致的间距

```css
/* ❌ 错误：随意的间距值 */
.bad-spacing {
    margin: 13px;
    padding: 17px;
}

.another-element {
    margin: 15px;
    padding: 23px;
}

/* ✅ 正确：使用一致的间距系统 */
:root {
    --space-1: 8px;
    --space-2: 16px;
    --space-3: 24px;
    --space-4: 32px;
}

.good-spacing {
    margin: var(--space-2);
    padding: var(--space-2);
}

.another-good-element {
    margin: var(--space-2);
    padding: var(--space-3);
}
```

## 5.8 总结

本章深入探讨了布局与网格系统的设计与实现：

1. **布局原则**：对齐、亲密性、重复、对比、空白、层级、平衡
2. **Flexbox**：一维布局，适合组件内部和简单页面布局
3. **Grid**：二维布局，适合复杂页面结构
4. **网格系统**：12列网格和8点网格系统
5. **常见布局**：侧边栏、圣杯、双飞翼、瀑布流等
6. **响应式**：移动优先的响应式设计策略
7. **最佳实践**：避免过度嵌套、使用语义化类名、保持一致性

掌握这些布局技术，你将能够构建灵活、可维护、响应式的现代Web界面。
