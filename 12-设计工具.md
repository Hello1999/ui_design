## 设计软件

### Figma（推荐）

**官网：** figma.com

**特点：**
```
✓ 基于浏览器，跨平台
✓ 实时协作
✓ 强大的组件系统
✓ 自动布局（Auto Layout）
✓ 原型功能
✓ 开发者友好
✓ 免费版功能完整
```

**核心功能：**

1. **组件（Components）**
```
- 创建可复用组件
- 变体（Variants）管理不同状态
- 实例（Instance）继承主组件
- 组件属性可覆盖
```

2. **自动布局（Auto Layout）**
```
- 自动调整容器大小
- 响应式设计
- 间距和对齐自动化
- 类似Flexbox
```

3. **设计系统**
```
- 颜色样式
- 文本样式
- 效果样式（阴影、模糊）
- 组件库共享
```

4. **原型**
```
- 交互流程
- 动画过渡
- 设备预览
- 分享链接
```

**快捷键：**
```
V - 选择工具
F - 框架
R - 矩形
O - 椭圆
T - 文本
K - 缩放
Cmd/Ctrl + D - 复制
Cmd/Ctrl + G - 编组
Cmd/Ctrl + / - 搜索菜单
Shift + A - 自动布局
```

**插件推荐：**
```
Unsplash - 免费图片
Iconify - 图标库
Content Reel - 虚拟内容
Contrast - 对比度检查
Stark - 可访问性检查
Figmotion - 高级动画
```
